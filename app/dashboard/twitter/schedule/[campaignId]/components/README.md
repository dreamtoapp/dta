# Components

This folder contains all UI components for the Campaign Posts page.

## Components:

- **StatusOverview.tsx** - Displays campaign statistics with colored status cards
- **PostsFilter.tsx** - Search and filter controls for posts
- **PostCard.tsx** - Main post card component with edit/view/send functionality
- **DeletePostDialog.tsx** - Confirmation dialog for deleting posts
- **ViewPostDialog.tsx** - Full post details dialog
- **WarningDialog.tsx** - Warning confirmation dialog before sending posts

All components follow shadcn/ui design patterns and are fully responsive.
